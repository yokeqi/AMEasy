#### 功能清单
* 去网络化，使用过程0网络交互，最大程度保证安全。
* 所有敏感数据均采用加密处理，数据更安全。
* 列表展示采用密码遮罩效果，例如 `123456` 显示为 `1***6`
* 基础的登录、增删改查功能。
* 支持账号分组、搜索功能。
* 支持超链接一键跳转。
* 基于.NET 4.6开发，Windows 10系统上可以直接运行。
