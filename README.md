# 账密宝 · 一款极致精简的账号密码管理软件

* > 互联网时代，每个人账号及相关的注册信息何其多，没有一个好用的账号管理工具来辅助管理，有谁能记得住。
* > 权衡点：如果网络化又担心安全性，如果本地化又牺牲了便捷性。
* > 小米MIUI 11新出了账号密码管理功能，可以试试看。
* > 超过20星开源

#### 功能清单
* 登陆 - 身份认证功能，支持修改用户资料。
* 所有交互数据均采用加密传输，还支持Https，数据更安全。
* 支持账号的增删改查操作。
* 列表展示支持密码遮罩效果，例如 `123456` 显示为 `1***6`
* 支持按 `标题`、`用户名`、`关键字` 模糊搜索。
* 支持账号分组功能。
* 支持超链接一键跳转，~~一键登录~~[^1]。
* ~~支持跨平台部署。~~[^2]
